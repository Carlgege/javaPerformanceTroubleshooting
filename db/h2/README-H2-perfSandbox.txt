perfSandbox H2 Installation Instructions
-----------------------------------------

1) Download the latest version of H2 from http://www.h2database.com/
2) Unzip h2.
   # cd perfSandbox/db
   # jar xvf h2-2016-01-21.zip
3) The start, stop, and load scripts in perfSandbox/db will only work if
      the following H2 files are unzipped to the same folder as this text file (README-H2-perfSandbox.txt).

	build.sh
	build.bat
	src
	service
	docs
	bin
